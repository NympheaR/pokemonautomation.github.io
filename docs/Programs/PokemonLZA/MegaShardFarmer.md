# Mega Shard Farmer

## Program Description

Farm Mega Shards in the alley behind Restaurant Le Yeah.

<img src="images/MegaShardFarmer.png">

## Instructions

**Switch Settings:**

1. Screen size: Must be 100% within the Switch settings
2. [Switch 2: All HDR options must be disabled.](../NintendoSwitch/Switch2Notes.md#switch-2-hdr-may-be-problematic)

**Program Settings:**

1. Video Resolution: 1080p or higher

**Setup:**

1. Your Camera controls should be set to default - Regular (not Inverted) with Camera Sensitivity 3.
2. The bottom move of your selected Pok√©mon is a ranged attack capable to hitting shards high off the ground.
3. Make sure you have unlocked Restaurant Le Yeah as a fly spot.
4. Start the program in front of Restaurant Le Yeah.
